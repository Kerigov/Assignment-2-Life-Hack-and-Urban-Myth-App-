# Life Hack or Urban Myth?

---

## App Overview

**Life Hack or Urban Myth?** is a native Android flashcard quiz application developed using **Kotlin in Android Studio**.

The app challenges users to distinguish between **real-life productivity hacks** and **misleading urban myths**, promoting critical thinking and digital literacy in an engaging, gamified format.

---

##  Purpose of the App

This app helps users:

* Identify **true life hacks**
* Recognise **false myths**
* Improve **critical thinking skills**
* Learn through **interactive feedback**

---

## ✨ Features

### ✅ Core Features

* Welcome screen with introduction and Start button
* Flashcard-style quiz with multiple questions
* "Life Hack" and "Urban Myth" answer options
* Instant feedback after each answer
* Score tracking throughout the quiz
* Final score screen with personalised feedback

### 🚀 Advanced Features (for high marks)

* Loop-based question system
* Data class for structured question storage
* Intents for navigation between screens
* Clean UI following Material Design principles
* Reset functionality ("Play Again")

### 📊 Bonus Feature

* Review screen showing:

  * All questions
  * Correct answers
  * Explanations
